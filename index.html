Place Cards in Matrices
Make both matrices have equal sums in their matching rows or columns.
Matching rows or columns with equal values get removed from the game
Game Concept By:

Ryan Herwig,
Mary Larsen,
Joseph Gross,
Darius Herman


Game Coded By:

Ryan Herwig
Card Name All Face Cards 10 Values Individual Values
Ace 1 or 11, able to be swapped throughout the game 1 or 14, able to be swapped throughout the game
Jokers 0 0
Jacks 10 11
Queens 10 12
Kings 10 13
All numbered cards have a value of their numerical value

Victory Condition:
Getting all cards out of your hand

Lose Condition:
All matrices get filled and no possible moves remain
